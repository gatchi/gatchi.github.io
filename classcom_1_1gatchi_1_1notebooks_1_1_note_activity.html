Detailed Description

Manages the note edit screen.

Includes:

  • New notes creation,
  • Updating existing ones,
  • Deleting existing ones if text is changed to blank
  • Text formatting
  • Drawing

◆ onBackPressed()

void com.gatchi.notebooks.NoteActivity.onBackPressed ( )
inline

Makes pressing back button save note.

Normally (I believe) hitting the back button exits the current Activity in Android and returns to the last. I think it's a bit more complicated than that, but regardless, this method overrides that behavior and forces a save so that no one loses a note edit from pressing back. Plus, it may be more intuitive for many users to use the back button to save and exit than hitting the dedicated save & exit button.
